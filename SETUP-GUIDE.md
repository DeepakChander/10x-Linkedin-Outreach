## ğŸ› Troubleshooting

### Setup doesn't run automatically

**Check:**
```bash
cat .claude/SETUP_CHECK.md
```

**If it says "COMPLETE" but you want to re-run:**
```bash
rm .claude/SETUP_CHECK.md
```

### Python dependencies fail to install

**Try:**
```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

### Node dependencies fail to install

**Try:**
```bash
cd tldraw-canvas
npm cache clean --force
npm install
```

### Statusline not showing

**Check `.claude/settings.json`:**
```json
{
  "statusLine": {
    "type": "command",
    "command": "node .claude/statusline.cjs"
  }
}
```

**Make sure `node` is in PATH:**
```bash
node --version
```

### Environment variables not working

**Check `.env` file exists:**
```bash
ls .env
```

**Check format (no quotes around values):**
```env
EXA_API_KEY=abc123
```
NOT:
```env
EXA_API_KEY="abc123"
```
